**Responsible Disclosure Policy:**

If you discover a security vulnerability:

1. **DO NOT** post it publicly on GitHub or social media
2. **DO** email [security@codekaiju.com](mailto:security@codekaiju.com) with:
   - Description of the vulnerability
   - Steps to reproduce
   - Proof of concept (if safe)
   - Your contact information
3. Code Kaiju Studios will:
   - Acknowledge receipt within 48 hours
   - Assess and prioritize the issue
   - Provide a fix timeline
   - Credit you in release notes (if desired)
4. **DO NOT** disclose the vulnerability publicly until Code Kaiju Studios has patched it

### 7.3 Bug Bounty Program

Code Kaiju Studios may offer rewards for reported security vulnerabilities:

- **Critical Bugs:** Up to $5,000 USD
- **High Severity:** Up to $2,500 USD
- **Medium Severity:** Up to $1,000 USD
- **Low Severity:** Up to $250 USD

Bounty eligibility requires:

- Following responsible disclosure guidelines
- Not exploiting the vulnerability for personal gain
- Providing sufficient technical detail for reproduction
- Not attempting to breach other systems or user data
